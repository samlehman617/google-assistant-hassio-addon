# Google Assistant Hassio Addon


Modified version of the default Google Assistant addon.  


### Features:

- [x] Webserver to make RESTful calls to Assistant
- [ ] Custom hotword detection
- [ ] Bluetooth speaker support
- [ ] Custom conversation configuration



### Requirements:

- armhf/amd64 device
- Microphone
- Speaker
